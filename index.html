Instructions for downloading and installing
TitanArchersScoring App - on android devices

  • Step 1 - tap app-debug.apk

  • Step 2 - scrolldown, tap download icon

  • Step 3 - tap download

  • Step 4 - tap Open

  • Step 5 - tap Install

If App not Installed! message
you must allow chrome or (browser) permission to download unknown app
on Android 9

  • a - go to phone settings

  • b - find Apps & Notifications

  • c - find web Browser to use

  • d - Advanced

  • e - Install unknown apps

  • f - Verify browser will allow from this source

on Android 10

  • a - go to phone settings

  • b - find Apps

  • c - Advanced

  • d - Special app access

  • e - Install unknown apps

  • f - Verify browser will allow from this source

  • Might also get two messages from Play Protect upon installation

  • 1 - Must answer Install Anyways

  • 1 - Must answer Don't send

Else App installed message

  • Step 6 - tap Open
