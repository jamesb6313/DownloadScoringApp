Instructions for downloading and installing
TitanArchersScoring App - on android devices

on Android 10

  • a - Go to phone's Settings

  • b - Select Apps

  • c - Select Advanced

  • d - Select Special app access

  • e - Select Install unknown apps

  • f - f - Verify browser "Will allow from this source" is set to on

You might also get two messages from Play Protect upon installation

  • 1 - you must answer "Install Anyways" to allow installation to continue

  • 2 - you must answer "Don't send" to complete installation
