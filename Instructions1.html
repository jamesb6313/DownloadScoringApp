Instructions for downloading and installing
TitanArchersScoring App - on android devices

  • Step 1 - Go to my Github Repositories found here:
    https://github.com/jamesb6313

  • Step 2 - Go to TianArchersScoring Repository

  • Step 3 - Tap on View Code

  • Step 4 - Tap on app-debug.apk

  • Step 5 - Scrolldown

  • Step 6 - Tap on download icon

  • Step 7 - Tap on Open

    • If using Chrome you might get the following alert
    "For your security, your phone is not allowed to install unknown apps from this source"
    Then you must go to settings and change to allow installation

    • Step 8 - Tap on Install

    Android 9.0 - If App not Installed! message
    you must allow chrome or (browser) permission to download unknown app
    This can be done prior to download and installation. See: Add permission - Android 9.0

    on Android 10
    See: Add permission - Android 10.0

    Else App installed message

    • Step 9 - Tap on Open
