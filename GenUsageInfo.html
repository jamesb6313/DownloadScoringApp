Instructions for using
TitanArchersScoring App - on android devices

    • Upon initial launch a pop up will ask for user name and target distance (i.e. Details).
    • This information is used to record your score and is optional.
    • If it is not entered then it will appear again the next time the app is launched.
    • Else, if it is entered it wil be stored locally on your phone. It will be displayed on the saved score card and used the next time the app is launched.
    • This information can be changed later by using the "Detail" button.
    • Place and hold finger on target image and a virtual arrow point will appear, just above and to left of this point.
    • Also, the current score of this point is displayed in the bottom left of the target image.
    • While holding, move finger to position the virtual arrow point to register its score.
    • When satisfied with the placement and score remove finger.
    • The score will now register in the appropriate arrow box on the score card.
    • If positioning or score is incorrect it can be correct by using th "Redo" button (see below).
    • Continue to do this for all arrows shot. The arrows will be grouped into threes and have an associated color for the group.
    • As each score is registered on the score card a running total will be displayed, for each group of six arrows, in the last box at end of each line on the score card.
    • The Running total for all shots will be displayed at the bottom left of the score card and also in the apps heading.
  4. Using the Redo Button video
    • The Redo Button allows the deletion of the last arrow point.
    • The arrow point will only be removed from the target display once the next arrow point is started.
    • On repeated pressing of the Redo Button multiple points will be deleted until no points exist (i.e. start all over).
  5. Using the Detail Button video
    • The Detail Button allows the users name and the target distance to be recorder on the score card.
    • The distance can be in feet, yards or meters. To change the units use the dropdown box.
    • The information is not required, but if not entered the app will request it upon each start up.
  6. Using the Save Target Button video
    • The Save Target Button allows the target and score card to be saved as an image.
    • The image can be shared to other apps.
